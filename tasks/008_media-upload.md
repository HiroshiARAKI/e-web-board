# Task 008: メディアアップロード機能

## ステータス
- [ ] 完了

## 概要
管理画面から画像・動画をアップロードし、ボードのスライドショーに利用できるようにする。

## やること
- [ ] `POST /api/media` — メディアアップロード API
  - multipart/form-data でファイル受信
  - ファイルタイプのバリデーション (JPEG / PNG / WebP / GIF / MP4 / WebM)
  - ファイルサイズの上限チェック
  - `public/uploads/` へのファイル保存
  - DB へのメタデータ登録 (MediaItem)
- [ ] `DELETE /api/media/:id` — メディア削除 API (ファイル + DB レコード)
- [ ] `PATCH /api/media/order` — メディア表示順の一括更新 API
- [ ] 管理画面のメディアアップロード UI
  - ドラッグ&ドロップ対応
  - アップロード進捗表示
  - サムネイルプレビュー
- [ ] 管理画面の表示順並べ替え UI (ドラッグ&ドロップ)

## 完了条件
管理画面から画像/動画をアップロードでき、ボード表示画面のスライドショーで表示される。並べ替えが反映される。
