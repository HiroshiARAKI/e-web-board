# Task 009: リアルタイム更新 (Server-Sent Events)

## ステータス
- [ ] 完了

## 概要
管理画面や外部 API からの変更をボード表示画面にリアルタイムで反映するため、SSE (Server-Sent Events) を実装する。

## やること
- [ ] `src/lib/sse.ts` — SSE 管理ユーティリティ
  - ボード ID ごとの SSE チャネル管理
  - クライアント接続/切断の管理
  - イベント発行関数
- [ ] `GET /api/sse/:boardId` — SSE エンドポイント
  - ボード ID に対応するストリームを開く
  - Content-Type: text/event-stream
  - 定期的なハートビートで接続維持
- [ ] ボード表示画面に SSE クライアントを組み込み
  - `EventSource` でサーバーに接続
  - イベント受信時にデータを再取得してUIを更新
  - 接続断時の自動再接続
- [ ] 以下の操作時に SSE イベントを発行
  - ボード設定の更新
  - メディアの追加・削除・並べ替え
  - メッセージの送信・削除 (外部 API 含む)

## 完了条件
管理画面でボード設定やメディアを変更した際、ボード表示画面がリロードなしで即座に更新される。
