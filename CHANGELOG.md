## [1.2.0] - 2026-04-15

### 新機能

- **管理者 PIN 認証を追加** (#56、closes #53)
  - ダッシュボード全体を 6 桁 PIN で保護
  - 初回起動時の PIN 設定フロー（PIN 入力 → 確認 → メールアドレス登録）
  - セッション Cookie (24h TTL) によるログイン維持
  - IP ベースのレート制限（5 回失敗 / 24h でロック）
  - メール / 画面表示によるリセットリンク発行

- **呼び出し番号テンプレートを追加** (#54、closes #52/#53)
  - 発番号を大きく表示する番号呼び出し専用テンプレート
  - 番号サイズ・色・フォント・アニメーションをカスタマイズ可能

- **管理画面にダークモードを追加** (#63、closes #58)
  - システム準拠 / ライト / ダークの 3 択
  - `localStorage` で設定を不揮発保存
  - `prefers-color-scheme` 変更にリアルタイム追従
  - テーマ切替時の 300ms スムーストランジション

- **アプリ名を Keinage にリブランディング** (#67、closes #60)
  - アプリ全体のブランド名を「e-Web Board」→「Keinage」に変更
  - ランディングページにロゴ・スクロールアニメーション付きヘッダーを追加
  - LP に Inter フォントを適用
  - ファビコンを keinage.png に更新

### テンプレート・UI 拡張

- **全テンプレートにフォント選択を追加** (#50、closes #48)
  - 日本語対応フォント（DotGothic16、衡山毛筆フォント等）と英語フォント（Source Code Pro 等）を選択可能

- **シンプル掲示板ティッカーにフォントサイズ・縦位置オプションを追加** (#51、closes #49)

- **メディア表示に objectFit (cover / contain) 設定を追加** (#47、closes #46)

### 不具合修正

- **Docker build 時のプリレンダリングエラーを修正** (#62、closes #59)
  - ダッシュボード layout で `cookies()` を DB クエリより先に呼び出し
  - マイグレーション前でも `next build` が成功するように

### その他

- GitHub Issue テンプレートを追加 (#66、closes #65)
- AI コーディングエージェント向け指示ファイルを追加 (#61、closes #55)

## [1.1.0] - 2026-04-12

### 新機能

- **全テンプレートに時刻・天気表示オプションを追加** (#29)
  - シンプル掲示板: スライドショー右上にオーバーレイ表示
  - メッセージ掲示板: ヘッダーに時計、フッター上部に天気バー
  - レトロ掲示板: ヘッダー下に時計・天気バーを表示
  - 各テンプレートの設定エディタに showClock / showWeather トグルを追加（デフォルト OFF）

- **画像リサイズ・サムネイル自動生成** (#31)
  - アップロード時に画像の長辺を最大ピクセル数でリサイズ（デフォルト 3840px）
  - 600px サムネイルを自動生成し、ダッシュボードで使用
  - 管理者設定画面から最大値の変更が可能

- **メディアの個別削除機能** (#32)
  - 設定画面にアップロード済みメディア一覧をサムネイル付きで表示
  - ディスクベースのファイル管理（孤立ファイルの検出・削除にも対応）
  - 削除時にボード紐付け情報の警告ダイアログを表示

- **管理者設定画面にSWバージョン情報を表示** (#34)
  - 現在のバージョンをリリースノートへのリンク付きで表示
  - GitHub Releases API で最新バージョンを確認し、更新がある場合に通知

- **ボード表示画面に全画面表示・マウスカーソル自動非表示機能** (#37)
  - マウスカーソルを3秒操作なしで非表示にし、動かすと再表示
  - 左下に全画面表示 / 元に戻すボタンをフロート表示
  - Fullscreen API でアドレスバー・タブ等を非表示に

## [1.0.0] - 2026-04-12

### 初回リリース 🎉

Web 上で動作するカスタマイズ可能なデジタルサイネージ「Keinage」の初回リリースです。

### テンプレート

- **シンプル掲示板** — メディアスライドショー＋下部テキストティッカー
- **フォトクロック掲示板** — 全画面スライドショー＋日時・天気オーバーレイ
- **レトロ掲示板** — ドットマトリクス風デザイン、文字フリップアニメーション
- **メッセージ掲示板** — API 連携によるリアルタイムメッセージ表示

### ダッシュボード (管理画面)

- ボードの作成・編集・削除
- テンプレート選択およびテンプレート別設定
- メディア (画像・動画) のドラッグ＆ドロップアップロード
- メッセージの作成・編集・削除 (優先度・有効期限設定)
- 天気予報の対象地域設定

### API (外部連携)

- ボード CRUD (`/api/boards`)
- メディア管理 (`/api/media`)
- メッセージ送受信 (`/api/messages`) — 外部システムからの呼び出しに対応
- 設定取得・更新 (`/api/settings`)
- 天気情報取得 (`/api/weather`)

### リアルタイム通信

- Server-Sent Events (SSE) によるボード表示のリアルタイム更新
- ボード・メディア・メッセージの変更即時反映
- 自動再接続機能

### デプロイ

- Docker Compose によるワンコマンド起動
- SQLite による外部 DB サーバー不要な構成
- データの Docker ボリューム永続化

### 技術スタック

- Next.js 15 (App Router) / TypeScript
- Tailwind CSS v4 / shadcn/ui / Framer Motion
- Drizzle ORM + SQLite
- Zod バリデーション
