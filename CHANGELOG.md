# Changelog

このプロジェクトのすべての注目すべき変更はこのファイルに記録されます。

## [1.0.0] - 2026-04-12

### 初回リリース 🎉

Web 上で動作するカスタマイズ可能な電子掲示板ツール「e-Web Board」の初回リリースです。

### テンプレート

- **シンプル掲示板** — メディアスライドショー＋下部テキストティッカー
- **フォトクロック掲示板** — 全画面スライドショー＋日時・天気オーバーレイ
- **レトロ掲示板** — ドットマトリクス風デザイン、文字フリップアニメーション
- **メッセージ掲示板** — API 連携によるリアルタイムメッセージ表示

### ダッシュボード (管理画面)

- ボードの作成・編集・削除
- テンプレート選択およびテンプレート別設定
- メディア (画像・動画) のドラッグ＆ドロップアップロード
- メッセージの作成・編集・削除 (優先度・有効期限設定)
- 天気予報の対象地域設定

### API (外部連携)

- ボード CRUD (`/api/boards`)
- メディア管理 (`/api/media`)
- メッセージ送受信 (`/api/messages`) — 外部システムからの呼び出しに対応
- 設定取得・更新 (`/api/settings`)
- 天気情報取得 (`/api/weather`)

### リアルタイム通信

- Server-Sent Events (SSE) によるボード表示のリアルタイム更新
- ボード・メディア・メッセージの変更即時反映
- 自動再接続機能

### デプロイ

- Docker Compose によるワンコマンド起動
- SQLite による外部 DB サーバー不要な構成
- データの Docker ボリューム永続化

### 技術スタック

- Next.js 15 (App Router) / TypeScript
- Tailwind CSS v4 / shadcn/ui / Framer Motion
- Drizzle ORM + SQLite
- Zod バリデーション
